💰 Bankroll & Kelly
Calculadora Half-Kelly
Fórmulas
EV = p × cuota − 1
Prob. implícita = 1 / cuota
Margen casa = Σ(1/cuota) − 1
Kelly% = (p × b − q) / b  [b=cuota−1, q=1−p]
Half-Kelly = Kelly% / 2
Stake = Half-Kelly × Bankroll

¿Por qué Half-Kelly? El Kelly completo maximiza el crecimiento logarítmico pero es agresivo. Dividir entre 2 reduce la volatilidad preservando buena parte del crecimiento. Si el EV es negativo, Kelly devuelve un valor negativo — no apostar.

1. Valor Esperado (EV)

El Valor Esperado mide si una apuesta es rentable a largo plazo. EV positivo = ganancia esperada; EV negativo = pérdida esperada.

Ejemplo: prob. real = 55%, cuota = 2.10
EV = 0.55 × 2.10 − 1 = 1.155 − 1 = +0.155 (+15.5%)
Por cada €100 apostados, retorno esperado €115.50

La casa añade margen bajando las cuotas, por lo que la mayoría de apuestas tienen EV negativo. Solo apuesta cuando el modelo detecte valor positivo, con disciplina de bankroll.

2. xG y Distribución de Poisson

El xG (goles esperados) cuantifica la calidad de las ocasiones. Un xG de 2.0 significa que el equipo «merecería» ~2 goles con sus tiros, independientemente del resultado real.

Dado un xG λ, Poisson calcula la probabilidad de marcar exactamente k goles:

P(k | λ=1.8) = e⁻¹·⁸ × 1.8ᵏ / k!
P(0)≈16.5%  |  P(1)≈29.7%  |  P(2)≈26.7%  |  P(3)≈16.0%

EDGE2026 construye una matriz 10×10 con todos los marcadores posibles y calcula la probabilidad conjunta de cada resultado. La mezcla xG_local = xGf_local×0.65 + xGa_visitante×0.35 pondera el ataque propio (65%) y la defensa rival (35%).

3. Cómo elegir mercado
  • 1X2: Cuando la diferencia de xG es clara (≥0.5) y hay suficiente muestra.
  • Goles totales: Over si ambos xG ≥ 1.5; Under si ambos ≤ 1.0.
  • BTTS: Cuando ambos equipos tienen xGf ≥ 1.0 consistentemente.
  • Hándicap asiático: Reduce margen de casa en favoritos claros; -0.5 es el más limpio.
  • Córneres: Equipos de alta posesión generan más. Usa datos reales de API para mejor precisión.
  • Tarjetas: +10% en fases eliminatorias. Partidos de alta intensidad o con rivalidad.
  • Mercados 1ª mitad: Más volátiles. Útil si un equipo tiene historial de salida fuerte.
4. Gestión de Bankroll y Kelly

El Criterio de Kelly determina el % óptimo del bankroll a apostar para maximizar el crecimiento a largo plazo.

p=55%, cuota=2.10, b=1.10, q=45%
Kelly = (0.55×1.10 − 0.45) / 1.10 = 0.155/1.10 ≈ 14.1%
Half-Kelly = 7.05% del bankroll
  • Nunca superes el 5% por apuesta aunque Kelly lo permita.
  • EV negativo → Kelly negativo → no apostar.
  • Recalcula el bankroll tras cada resultado (es una estrategia dinámica).
  • Half-Kelly compensa la incertidumbre en tus estimaciones de probabilidad.
5. Fuentes de datos recomendadas
  • Sofascore — xG en vivo y estadísticas por partido
  • FBref — base de datos histórica profunda (xG, presión, etc.)
  • xGScore — xG específico por partido
  • Oddsportal — comparador de cuotas históricas
  • Betexplorer — resultados y cuotas apertura/cierre
  • WC2026 API — datos en tiempo real del torneo
